# Django Templates Assignment

This repository contains a simple Django project created as part of a Templates assignment.

## Features
- Django project named `project`
- Application named `pages`
- HTML template `index.html` displaying personal information:
  - Full Name
  - Student ID
  - Address
  - Email Address
  - Note about the assignment
- Styled using CSS with a responsive and clean card layout
- Hover effect on the card for a subtle interactive feel
